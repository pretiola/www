Welcome to My Site

This is a minimal Actix-web + Tera + Tailwind template. Edit this page to get started.

Getting Started

Drop your HTML content into the templates/ directory. Any .html file you add will automatically be routable at /{filename}.html and included in the sitemap.

Add images to static/pictures/ and run npm run build:images to generate optimized responsive variants.

Edit static/css/styles.css to customize your Tailwind styles, then run npm run build:css to compile.
